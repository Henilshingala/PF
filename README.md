A premium, cinematic dark-themed portfolio designed for high-performance engineering showcases. Built with Next.js 14, Framer Motion, and Tailwind CSS, this portfolio emphasizes clean architecture, security, and elite user experience.

---

## ✨ Key Features

### 🛠️ Engineering Foundation
- **Next.js 14 App Router**: Utilizing the latest React server components for optimal speed.
- **Dynamic Tech Ecosystem**: A custom-built, animated 3-layered orbit system showcasing 40+ technologies.
- **Glassmorphism UI**: Premium visual depth using backdrop blurs and subtle border glows.

### 📱 Performance & Responsiveness
- **GPU Accelerated**: Optimized animations using `will-change` and hardware-accelerated transforms to ensure 60fps even on mobile.
- **Fully Responsive**: Hand-crafted layouts that adapt perfectly from iPhone screens to 4K Ultrawide monitors.
- **Optimized Assets**: Automatic image conversion to WebP/AVIF via Next.js Image optimization.

### 🔒 Security & Best Practices
- **Hardened Links**: Secure `rel="noopener noreferrer"` for all external connections.
- **Custom Security Headers**: Integrated X-Frame-Options, Content-Type-Options, and Referrer Policies.
- **SEO Ready**: Full metadata configuration with OpenGraph and Twitter cards for professional social sharing.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Henilshingala/PF.git
   cd PF
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:3000` to view the portfolio.
